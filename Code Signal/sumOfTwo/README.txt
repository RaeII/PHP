You have two integer arrays, a and b, and an integer target value v. Determine whether there is a pair of numbers, where one number is taken from a and the other from b, that can be added together to get a sum of v. Return true if such a pair exists, otherwise return false.

Você tem duas matrizes de inteiros, a e b, um valor de destino inteiro v. Determine se há um par de números, de onde um número é retirado a e o outro b,  podem ser somados para obter uma soma de v. Retorne true se tal par existir; caso contrário, retorne false.
